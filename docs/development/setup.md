# Development Cleanup

## Prerequisites

- Go 1.21+
- Node.js 18+
- Docker
- Python 3.10+

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/arc-platform/monorepo.git
    cd monorepo
    ```

2.  **Run Setup Script:**
    ```bash
    ./scripts/development/setup.sh
    ```

3.  **Start Services:**
    ```bash
    docker-compose up -d
    ```
